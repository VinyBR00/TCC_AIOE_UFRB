# Algoritmo de Pontos Interiores em Elipsoides (AIOE) 📐

Este repositório contém o código-fonte em LaTeX e os recursos computacionais referentes ao meu Trabalho de Conclusão de Curso (TCC) em Matemática na **Universidade Federal do Recôncavo da Bahia (UFRB)**.

* **Status:** Em desenvolvimento (Defesa prevista para Julho/2026)

## 📌 Sobre o Projeto
O trabalho foca na análise e implementação do **Algoritmo de Pontos Interiores em Elipsoides (AIOE)**, explorando a convergência e eficiência de métodos de otimização aplicados a geometrias elipsoidais.
